# ゲーム仕様

## 概要

ターン制RPG REST API。キャラクターを作成し、ダンジョンを探索して敵と戦うゲームバックエンド。

## 機能一覧

| 機能 | 内容 |
|------|------|
| キャラクター作成 | 名前・職業選択 |
| ステータス | HP・MP・攻撃力・防御力・レベル・経験値 |
| ダンジョン探索 | 部屋の移動・イベント発生 |
| ターン制戦闘 | 攻撃・スキル・アイテム・逃げる |
| アイテム・装備 | インベントリ・装備による能力変化 |
| スキル・魔法 | 職業ごとのスキル習得 |
| ショップ | ダンジョン外でアイテム購入 |
| セーブ・ロード | DB永続化（セッションもDB保存） |

## キャラクター仕様

- 1アカウント1キャラクター
- 職業はDBマスタで管理（データ追加で拡張可能）
- レベルアップ時にステータスポイントを獲得し、自由に振り分け可能

## 職業仕様

- 職業ごとに初期ステータスが異なる
- 職業ごとに習得できるスキルが異なる（レベル条件あり）
- 職業・スキルの種類はDBマスタで管理

## ダンジョン仕様

- 複数のダンジョンが存在し、推奨レベルが設定されている
- 固定マップ（roomsテーブルで管理）
- 部屋タイプ：normal / boss / treasure / rest
- 最終部屋はボス固定
- 探索中・戦闘中の状態はDBに保存（ブラウザを閉じても再開可能）

## 戦闘仕様

- 部屋移動時に自動で戦闘開始（ドラクエ方式）
- アクション：attack / skill / item / run
- 戦闘勝利時：経験値・ゴールド・アイテムドロップ
- レベルアップ判定は戦闘結果レスポンスに含める
- 死亡時：街に戻る・HP1で復活・所持金半減

## 装備仕様

| 部位 | 説明 |
|------|------|
| weapon | 武器 |
| armor | 防具 |
| helmet | 兜 |
| shield | 盾 |
| accessory | アクセサリ |

## ショップ仕様

- ダンジョン外に設置
- ゴールドで購入

## レベルアップ仕様

- EXP閾値はDBテーブル（`level_exp_thresholds`）で管理
- レベルアップ時にステータスポイントを獲得
- レベルアップ時にスキル習得判定を実施
- レベルアップ情報は戦闘結果レスポンスに含める
